# Deeper Dungeons

Deeper Dungeons is Michael Holloway's code-learning sandbox.

  - It's in Python.
  - It's modular.
  - It's _fun!!!_

# New Features!

  - Monsters!
  - Treasure!
  - A Mystic!
  - Potions! Which almost work!
